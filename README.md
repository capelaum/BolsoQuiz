# BolsoQuiz 

![Capa do Projeto](/_docs/Capa-BolsoQuiz.jpeg)

Uma quiz sobre o presidente Jairr Bolsonaro utilizando Next.js e Plataforma da [Vercel][4] para deploy.

## Tecnologias

- [Next.js][1]
- [Styled Components][2]
- [React][3]
- [Vercel][4]

### Créditos

Projeto feito na Imersão React da [Alura][5] 2.
